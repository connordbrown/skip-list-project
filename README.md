# skip-list-project

## introduction

This project is a C++ implementation of a Skip List.

## operation

This application uses C++ 11 or later via CMake. Build it with the following commands:

```bash
cd build
cmake ..
```

Then, compile and run it:
```bash
make && ./run_app
```
